# Required deps
-   SDL2
-   SDL2_image
-   SDL2_ttf


### How to install
1. Windows
   - install MSYS2
   - open an MSYS2 MSYS window
   - type in ```pacman -S mingw-w64-x86_64-SDL2 mingw-w64-x86_64-SDL2_image mingw-w64-x86_64-SDL2_ttf```
2. Linux (Arch Linux)
   - open the console
   - type in ```sudo pacman sdl2 sdl2_image sdl2_ttf```
